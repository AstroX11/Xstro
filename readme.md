## SETUP

[Download Node.js](https://nodejs.org/en)

[Download Ffmpeg](https://www.ffmpeg.org/)

[Download Git](https://git-scm.com/)

```bash
git clone https://github.com/AstroX11/Xstro.git
```

```bash
cd Xstro
```

```bash
npm i -g yarn
```

```bash
yarn
```

```
yarn build
yarn start
```
